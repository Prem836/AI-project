# DocSensei

An AI-powered document question answering system that allows users to upload PDF and DOCX files and ask questions about their content.

## Features

- PDF Upload
- DOCX Upload
- AI Question Answering
- Document Summarization
- RAG
- FAISS
- Gemini

## Tech Stack

- Streamlit
- LangChain
- FAISS
- Google Gemini
- Hugging Face
